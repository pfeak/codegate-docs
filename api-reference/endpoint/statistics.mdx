---
title: "获取统计信息"
openapi: "GET /api/v1/projects/{project_id}/statistics"
---

获取指定项目的统计信息：`total_codes`、`used_codes`、`unused_codes`、`disabled_codes`、`expired_codes`、`usage_rate` 及 `recent_verifications`（最近核销记录）。与 [获取项目信息](/api-reference/endpoint/get-project) 中的 `statistics` 字段一致，本接口提供更完整的近期核销列表。
